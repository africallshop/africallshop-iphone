\subsection{Detailed Description}
The \doxyref{Linphone\-Call}{p.}{group__call__control_gac0a4861d7c4702c64f7106f2b893c69e} object represents an incoming or outgoing call managed by the \doxyref{Linphone\-Core}{p.}{group__initializing_gaa5cf635b82dd338e7ee2dd3599d05f0a}. Outgoing calls can be created using \doxyref{linphone\-\_\-core\-\_\-invite()}{p.}{group__call__control_gaaea662739cc3825eaa80d8ac264a883c} or \doxyref{linphone\-\_\-core\-\_\-invite\-\_\-address()}{p.}{group__call__control_gaac6720bef666db932ae77e07c2201b4c}, while incoming calls are notified to the application through the \doxyref{Linphone\-Core\-V\-Table\-::call\-\_\-state\-\_\-changed}{p.}{struct__LinphoneCoreVTable_a771c633cfce7bae2cc0160fbd377bf28} callback.

See the basic call \doxyref{tutorial}{p.}{group__basic__call__tutorials}.
